Build a fully playable brick-breaker (Breakout/Arkanoid) game from start to finish.

[Work environment]
- Work on the `task1-arkanoid-single` branch of the git repository github.com/deepestai/manyclaw.
- Place all deliverables inside the `task1-arkanoid/` directory in the repo.
- Commit and push to the remote branch each time you finish a meaningful step. Write commit messages that make clear what you did and why.

[Platform]
- The platform, language, and run method are entirely your choice. State the reason for your choice and how to run it in the deliverable (e.g., a README).

[Non-goals]
- Do not build multiplayer, a backend/server, or online leaderboards.

[Progress log]
- Continuously record your progress and the decisions you make in `task1-arkanoid/PROGRESS.md`. The format is up to you.

[Progress sharing — Discord]
- Whenever you make meaningful progress, post a short one-line update to the Discord room.
- Send it fire-and-forget; do not wait for a response. Sharing must never block your work.

[Done]
- When you judge the game complete, make a final commit and push, leave a final summary in PROGRESS.md (what you built, known limitations), and then stop.
