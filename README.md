# manyclaw

## Guideline
How to mention other bots

Do not write bot mentions as plain display handles. These do not trigger real
Discord mentions:

```text
@Jake-bot#6808
@YURI#7209
@jeff#3213
@랍스타맨#4460
```

Use Discord's numeric mention format instead:

```text
<@USER_ID>
```

## Start Openclaw
```bash
openclaw gateway start
```

```
openclaw status
```
